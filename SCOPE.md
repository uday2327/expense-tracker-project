# Scope

## Included

- JWT registration and login.
- Date-aware group membership.
- Expense CRUD with soft delete.
- Equal, exact, percentage, and share-based splits.
- INR/USD storage with static USD conversion.
- Balance summaries, drill-downs, and simplified debts.
- CSV import with anomaly review before commit.
- Import anomaly audit records and downloadable JSON reports.
- Settlements and balance history.

## Out Of Scope For The First Local Version

- Live currency exchange APIs.
- NoSQL storage.
- Hard deletes.
- Session-cookie authentication.
